# Metro College of Technology - Better Jobs Ontario Funding

This project is a landing page for the Metro College of Technology, focusing on the Better Jobs Ontario Funding.

## Project Structure

- `index.html`: The main HTML structure of the landing page.
- `style.css`: External stylesheet containing all the visual styling and embedded assets.

## How to Run

Simply open `index.html` in any modern web browser.

## Refactoring Note

The project has been refactored to separate HTML and CSS for better maintainability and to follow best practices.
